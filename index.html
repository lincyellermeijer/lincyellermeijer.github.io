  • Fasten Your Seatbelts

    • Fasten Your Seatbelts is about designing and creating a simple IT-system. With this project students will come in contact with java and processing. They learn how to convert the wishes of a client into clearly defined requirements and build a IT-system that meets these requirements.

      This was my first project when I started my study at the HvA. I worked together with 6 other students from Game Development to create a game. At first we had to think of 3 different game concepts, but in the end we chose to elaborate on this one, "Prehistoric Defender". This project was in association with Corendon.

  • Students in Motion

    • As a propedeuse student from the domain Digitail Media and Creative Industry (DMCI) from the HvA we participated in Student in Motion. The theme was "the new making". We've collaborated with craft entrepreneurs in Amsterdam, to support them with our DMCI-expertise.

      This is a yearly organized project that takes places during one week in February. I participated in this project from 02-02-2015 'till 06-02-2015.

  • Project Agile Development

    • Project Agile Development is about creating a IT-system with a multidisciplinair team. With this project students will learn to work with the Agile Development method 'Scrum'. Futhermore, they learn how to model a system with the use of the modeling language 'UML'.

      In Project Agile Development I worked together with 5 other students from different ICT studies. We build an app (in the form of a single-page website with multiple views) with HTML, AngularJS, CSS and JavaScript. It was in association with the Waag Society. The app is developed for elderly people recovering from a hip procedure, the goal of the app is to stimulate rapid rehabilitation.

  • Project Game Development

    • During Project Game Development students create a game in teams. All kinds of game genres are permitted, as long as it is for use in the entertainment market. In this project students will create several prototypes in a very short time and need to elaborate on one of these. Students learn to make better game design choices by gameplay tests and reports.

      In project Game Development I've worked together with 5 Game Development students to create a game. We've participated in 3 gamejams and decided to elaborate 1 of our prototypes into a fully functional game, "Boomerang Boy". We've worked on this project for a half year and developed a very nice indie platformer game in Unity with C#.

  • Project Mobile Development

    • During Project Mobile Development students develop one or more mobile applications for an extern client.

      In project Mobile Development I'm working together with 4 students from different ICT-studies to create an application. This is in association with Microsoft.
