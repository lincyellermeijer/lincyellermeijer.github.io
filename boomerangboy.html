Boomerang Boy

Information

In project Game Development I've worked together with 5 Game Development students to create a game. We've participated in 3 gamejams and decided to elaborate 1 of our prototypes into a fully functional game, "Boomerang Boy". We've worked on this project for half a year and developed a very nice indie platformer game for pc. I've used Unity to create the game, Visual Studio to write the C# code in and Paint.net to edit the assets/game art.

Overview

  • Tools: Unity, Visual Studio, Paint.net
  • Language: C#
  • Platform: PC
  • Duration: 6 months
  • Role: Game Designer, Gameplay Programmer
